# Finance Tracker — Frontend

A modern personal finance dashboard built with React, TypeScript, and Tailwind CSS.

## Live Demo
🌐 https://finance-tracker-frontend-aryan-2409s-projects.vercel.app

## Backend API
https://finance-tracker-api-1-uvuw.onrender.com

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS
- React Query (data fetching + caching)
- React Hook Form + Zod (form validation)
- Recharts (dashboard charts)
- Axios (API client)
- React Router (navigation)

## Features
- JWT authentication (login + register)
- Dashboard with income/expense charts
- Full transaction management (add, filter, delete)
- Category management
- Fully responsive layout

## Run Locally
```bash
npm install
echo 'VITE_API_URL=https://finance-tracker-api-1-uvuw.onrender.com/api' > .env
npm run dev
```
